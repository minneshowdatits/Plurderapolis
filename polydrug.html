💊 Rx + Party Drug Warnings

SSRIs / SNRIs

  • + MDMA → flat high, redosing, serotonin syndrome
  • + Psychedelics → unpredictable or muted
  • + Stimulants → anxiety, BP spikes

Wellbutrin (Bupropion)

  • + MDMA/Coke → seizure risk, panic
  • + Alcohol → aggression, blackout

Benzos

  • + Alcohol/Opiates → respiratory failure
  • + K/GHB → unconscious vomiting, blackout

Gabapentin / Lyrica

  • + Alcohol/Opiates → deep sedation, overdose

ADHD Meds

  • + MDMA/Coke → serotonin syndrome, heart risk
  • + Alcohol → blackout risk
  • + Psychedelics → destabilization

Antipsychotics

  • + Psychedelics → muted trips, psychosis
  • + MDMA → dull effects, redosing

Lithium

  • + Psychedelics → seizures, death

Hydroxyzine / Trazodone

  • + Alcohol / Benzos → sedation, overdose
  • + K/GHB → vomit risk, unconsciousness

🛟 Golden Rules

  • Don’t stack downers. (Xanax + alcohol = coma fuel)
  • Don’t stack uppers. (Adderall + coke = heart attack risk)
  • Test everything. Pills lie. Powders lie. Be sure.
  • Psychedelics aren’t just visuals—they shift your mind
  • Always have a trusted friend with you
  • Start low—especially if you’re on meds or recently sober
